WireGuard VPN Setup

WireGuard provides secure remote access to your homelab from anywhere. Choose your platform below for setup instructions.

Android Windows Linux macOS

Where are my config files?

WireGuard peer configurations are automatically generated when the container starts. Find them on your homelab server at:

./config/wireguard/peer_<name>/peer_<name>.conf

Each peer folder also contains a QR code image (peer_<name>.png) for easy mobile setup.

Show QR Code in Terminal

To display a peer's QR code directly in your terminal:

docker compose exec wireguard /app/show-peer <peer_name>

Replace <peer_name> with the peer name (e.g., phone, laptop).
