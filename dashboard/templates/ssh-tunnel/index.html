SSH Tunnel Access

Some services (like the i2pd web console) are bound to localhost for security. Use an SSH tunnel to access them securely from your local machine.

Linux macOS Windows

Why SSH Tunnels?

SSH tunnels provide secure, encrypted access to services that are intentionally not exposed to the network. This is more secure than opening ports directly.

When you create a tunnel, your local machine connects through SSH and forwards a local port to the remote service.

i2pd Console 7070 I2P router status and configuration
